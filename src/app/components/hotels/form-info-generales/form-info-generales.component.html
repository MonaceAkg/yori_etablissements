Où se situe l'établissement que vous inscrivez

thumb_up Que dois-je indiquer dans mon adresse?

  • Indiquez le numéro et le nom de rue du bâtiment.
  • Le numéro de l'étage dans la deuxième ligne de l'adresse, le cas échéant
  • Vous pourrez communiquer plus tard les numéros d'appartement ou d'étage.
  • Le code postal
  • Veillez à utiliser l'orthographe exacte pour le nom de la rue
  • Utilisez l'adresse postale de votre hébergement et non celle de votre bureau ou de votre domicile

light_mode Pourquoi dois-je indiquer mon adresse?

Une fois qu'un client réserve votre
hébergement, cette adresse lui sera
fournie. Il est essentiel qu'elle soit correcte
afin que le client puisse trouver facilement
votre hébergement.
